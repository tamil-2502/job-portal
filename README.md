# JobPortal — React Job Search Portal

A full-featured job search portal built with React, Tailwind CSS, and React Hook Form.

## 🔗 Live Demo


## ✨ Features
- Browse and search job listings with filters
- Apply for jobs with form validation
- Employer dashboard to post and manage jobs
- Applicant status management (Reviewing, Shortlisted, Hired)
- Role-based auth (Job Seeker / Employer)
- Dark mode support
- Mobile responsive with hamburger menu
- Data persistence with localStorage

## 👤 Demo Login
- Job Seeker — browse and apply for jobs
- Employer — post jobs and manage applicants
No real credentials needed, just select your role!
